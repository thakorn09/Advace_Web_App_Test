โจทย์ส่วนที่ 1 (14 คะแนน)
  • (1 คะแนน) Student ID : สามารถนำข้อมูลรหัส นศ. จากพร็อพเพอร์ตี้ profile มาแสดงได้
  • (1 คะแนน) Name : สามารถนำข้อมูลชื่อจากพร็อพเพอร์ตี้ profile มาแสดงได้
  • (1 คะแนน) Gender : สามารถนำข้อมูลเพศจากพร็อพเพอร์ตี้ profile มาแสดงได้
  • (1 คะแนน) Birth Year : สามารถนำข้อมูลปี คศ. เกิดจากพร็อพเพอร์ตี้ profile มาแสดงได้
  • (2 คะแนน) Age : สามารถนำข้อมูล Age ที่ได้จากการคำนวณปี คศ. เกิด (Birth Year) มาแสดงได้
  • (1 คะแนน) Email : สามารถนำข้อมูลอีเมล์จากพร็อพเพอร์ตี้ profile มาแสดงได้
  • (1 คะแนน) Phone : สามารถนำข้อมูลเบอร์มือถือจากพร็อพเพอร์ตี้ profile มาแสดงได้
  • (2 คะแนน) Address : สามารถนำข้อมูลที่อยู่จากพร็อพเพอร์ตี้ profile มาแสดงในรูปแบบ Alert เมื่อคลิกที่ปุ่ม show address ได้
  • (2 คะแนน) Comment : สามารถกรอกข้อมูล และแสดงข้อมูลนั้น ๆ ออกที่หน้าจอได้ทันที ดังตัวอย่าง
  • (2 คะแนน) รูปภาพฝั่งซ้ายมือ : สามารถนำ url ของรูปภาพจากพร็อพเพอร์ตี้ profile มาแสดงได้
